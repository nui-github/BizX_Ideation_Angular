# BizX — Design Prototype (AI Studio)

React + Vite prototype used for design ideation. **This is not the shipped app** — it's the
visual/behavioral reference for the Angular build in [`/angular`](../angular). See
[`/docs/HANDOFF.md`](../docs/HANDOFF.md) for how the two relate.

View in AI Studio: https://ai.studio/apps/83a9481d-2220-4906-9841-ab4452eb9515

## Run locally

**Prerequisites:** Node.js

1. Install dependencies: `npm install`
2. (optional) Set `GEMINI_API_KEY` in `.env.local` for AI Studio's Gemini integration
3. Run the app: `npm run dev`
